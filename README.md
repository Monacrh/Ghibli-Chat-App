### MY FIRST WEB CHAT


### Clone this project:

```
git clone https://github.com/Monacrh/Ghibli-Chat-App.git
```

### Navigate to the project directory:

```
cd Ghibli-Chat-App
```

### Get the dependency

```
npm install
```

### Start the server

```
npm run dev
```
